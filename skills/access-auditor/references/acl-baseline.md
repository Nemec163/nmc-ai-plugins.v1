# ACL baseline

Recommended levels:

1. New specialist: `A1_worker`
2. Domain builder: `A2_domain_builder`
3. System operator: `A3_system_operator`
4. Full orchestrator only: `A4_orchestrator_full`

Audit checklist:

1. Is write access broader than needed?
2. Is promote capability justified?
3. Is global facts access necessary?
